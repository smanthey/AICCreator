# OpenClaw Setup macOS App (Starter)

This is a SwiftUI starter app for packaging OpenClaw installation/configuration as a native macOS experience.

## What it does

- Guides user through install prerequisites
- Runs `install-openclaw-macos.sh`
- Captures Telegram token + basic settings
- Opens creator workflows dashboard links

## Build (Xcode)

1. Open Xcode and create a new **App** project named `OpenClawSetupApp`.
2. Replace generated `ContentView.swift` with `OpenClawSetupApp.swift` from this folder.
3. Add app sandbox permissions if you execute shell scripts.
4. Build + export `.app`.

## Runtime dependency

Use installer script generated by creator-pack agent:

- `artifacts/openclaw-creator-pack/*/scripts/install-openclaw-macos.sh`
