## 🔗 Configure Webhooks

### 1. Meta Business Manager (WhatsApp)

1. Go to: **Meta Business Manager** → **WhatsApp** → **Configuration** → **Webhook**
2. Set **Callback URL**: `https://autopayagent.com/webhooks/whatsapp`
3. Set **Verify Token**: `83b6bca4b79b4e535b2d3bc14`
4. Subscribe to: `messages`, `message_deliveries`, `message_reads`
5. Click **Verify and Save**

### 2. Stripe Dashboard

1. Go to: **Stripe Dashboard** → **Developers** → **Webhooks**
2. Click **Add endpoint**
3. Set **Endpoint URL**: `https://autopayagent.com/webhooks/stripe`
4. Select events:
   - `checkout.session.completed`
   - `checkout.session.async_payment_succeeded`
5. Copy the **Signing secret** and ensure it matches `STRIPE_WEBHOOK_SECRET` in your `.env`

## 🐛 Troubleshooting

### Server won't start
- Check all environment variables are in `.env`
- Verify port 3031 is not in use: `lsof -i :3031`

### WhatsApp messages not received
- Verify webhook URL is accessible: `curl https://autopayagent.com/webhooks/whatsapp?hub.mode=subscribe&hub.verify_token=83b6bca4b79b4e535b2d3bc14&hub.challenge=test`
- Check server logs for errors
- Ensure webhook is verified in Meta Business Manager

### Payments not confirming
- Check Stripe Dashboard → Webhooks → Recent events
- Verify `STRIPE_WEBHOOK_SECRET` matches Stripe
- Check server logs for webhook errors

### Messages not sending
- Verify `WHATSAPP_ACCESS_TOKEN` hasn't expired
- Check you're within 24-hour free messaging window
- Review WhatsApp API response in server logs
